# Autopsy Forensic Analysis – M57.biz (Jean Case)

## 📌 Project Overview

This project presents a digital forensic investigation of the **M57.biz (Jean Case)** using the Autopsy tool. The case simulates real-world cybercrime analysis involving user activities, web browsing, and email evidence.

## 🛠️ Tools Used

* Autopsy Forensic Tool
* Sleuth Kit
* Windows OS

## 🔍 Key Analysis Performed

* File system analysis
* Web activity examination
* Email analysis
* Keyword search
* Timeline analysis

## 📂 Dataset

* M57.biz forensic image (Jean Case dataset)

## 📊 Findings

* Identified user activities and browsing history
* Recovered important files and emails
* Detected potential suspicious actions

## 🎯 Conclusion

The investigation demonstrates how digital forensic tools like Autopsy can be used to collect and analyze evidence, helping in solving cyber-related cases.
